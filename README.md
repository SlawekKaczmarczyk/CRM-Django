# CRM-Django

Simple CRM project created using Django.

#Run Locally

```bash
git https://github.com/SlawekKaczmarczyk/CRM-Django
```

```bash 
cd CRM
```

Install dependencies
```bash
#Windows
pip install -r requirements.txt
#Linux
pip3 install -r requirements.txt
```

Start the server

```bash
#Windows
python main.py
#Linux
python3 main.py
```
